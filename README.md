## Idle Bar Game (Next.js)

Een simpele Next.js idle game waarin je bier tapt, verkoopt en je bar kunt upgraden.

### Scripts

- **npm install** – dependencies installeren
- **npm run dev** – development server starten op `http://localhost:3000`

### Idee

- Over tijd wordt automatisch bier ingeschonken.
- Je verkoopt glazen voor geld.
- Upgrades beïnvloeden onder andere:
  - tapsnelheid
  - hoeveelheid bier per tick
  - prijs per glas
  - automatische verkopers
- Er is alvast een basis-moreel-systeem ingebouwd dat invloed heeft op je efficiency. Je kunt dit later uitbreiden met meer events en keuzes.
